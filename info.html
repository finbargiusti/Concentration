What is Concentration?

Concentration is a system for students and the self-employed to concentrate by blocking the sites they usually get distracted until they have finished by tasks. It was made by Finbar Giusti with the intention of entering it into the BT Young Scientist And Technology Exibition.

How was it built?

It is written in C++ with Qt libraries for graphics.

Finbar wrote the system on his Linux laptop in 2017. Generally during the summer. He wrote the code , commited and pushed it to Github as often as he could.

How can I help?

You could add some of your own code and make a pull request on the Github bage. You can also use it and report any bugs in the Github issues page or just send me ideas on my Facebook page.

What's planned for the future?

In the future, the app will be ported to Android and also many features such as document wordcount detection are going to be added.
